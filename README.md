## Key Features

### Board Management
- Create, update, and delete boards
- Archive boards for later reference
- Mark boards as favorites
- Filter and search boards

### Column Management
- Add, edit, and delete columns
- Reorder columns with drag & drop
- Customize column names and colors

### Card Management
- Create cards with titles and descriptions
- Move cards between columns
- Reorder cards within columns
- Update and delete cards

### Error Handling
- Graceful error messages when backend is unavailable
- Toast notifications for user actions
- Comprehensive error boundaries
